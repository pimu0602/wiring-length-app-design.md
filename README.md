# 配線長算出アプリ

装置図面PDFをローカルのブラウザ内で開き、ユーザーが図面上に配線ルートを指定して、区間長・余長・推奨カット長を算出する静的Webアプリです。

## 使い方

1. `index.html` をブラウザで開く。
2. `PDF読み込み` から図面PDFを選択する。
3. `基準寸法設定` を押し、図面上の既知寸法の始点と終点をクリックする。
4. 実寸mmを入力して、PDF上の距離からmm換算できる状態にする。
5. `新規ルート` を押し、図面上をクリックして始点・経由点・終点を追加する。
6. 必要に応じて区間一覧の `採用長 mm` を手入力で上書きする。
7. 余長と切り上げ単位を確認し、`ルート確定` を押す。
8. 必要な配線ルートを作成したら `CSV出力` でカット長リストを保存する。

## 実装範囲

- PDF読み込みと表示
- ページ切り替え
- ズーム指定
- 表示幅に合わせる
- ページ全体表示
- 外側コンテナでの上下左右スクロール
- Shift + ホイールの横スクロール
- Ctrl + ホイールのズーム
- 操作モード切り替え
- 図面上クリックによる点追加
- 点同士の線表示
- 点のドラッグ移動
- 右クリックまたはボタンによる点削除
- 基準寸法設定
- 基準寸法からの自動計算長算出
- 区間一覧生成
- 区間長の手入力
- 余長入力
- 切り上げ単位設定
- 推奨カット長計算
- 複数ルート管理
- CSV出力
- LocalStorage保存

## 注意

PDFファイル自体は保存しません。作業を再開する場合は同じPDFを再度読み込んでください。点座標、区間長、余長、配線名、切り上げ単位はLocalStorageに保存されます。
