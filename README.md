# 配線長算出アプリ

装置図面PDFをローカルのブラウザ内で開き、ユーザーが図面上に配線ルートを指定して、区間長・余長・推奨カット長を算出する静的Webアプリです。

## 使い方

1. ローカルサーバーでこのフォルダーを配信する。
   例: `python -m http.server 8766`
2. ブラウザで `http://localhost:8766/` を開く。
3. `PDF読み込み` から図面PDFを選択する。
4. A3横図面は自動で横向き判定され、初期表示は `表示幅に合わせる` になる。
5. `基準寸法設定` を押し、図面上の既知寸法の始点と終点をクリックする。
6. 実寸mmを入力して、PDF上の距離からmm換算できる状態にする。
7. `新規ルート` を押し、図面上をクリックして始点・経由点・終点を追加する。
8. 点を間違えた場合は `元に戻す`、`最後の点を削除`、または点を選択して `選択点削除` を使う。
9. 必要に応じて区間一覧の `採用長 mm` を手入力で上書きする。
10. 余長と切り上げ単位を確認し、`ルート確定` を押す。
11. 必要な配線ルートを作成したら `CSV出力` でカット長リストを保存する。

## 注意

PDFファイル自体は保存しません。作業を再開する場合は同じPDFを再度読み込んでください。点座標、区間長、余長、配線名、切り上げ単位はLocalStorageに保存されます。

PDF.jsのES ModulesとWorkerを使うため、`file://` で直接開くのではなくローカルサーバー経由で使ってください。PDFは外部に送信されず、ブラウザ内で処理されます。
